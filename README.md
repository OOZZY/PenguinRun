# PenguinRun

A third-person racing-like game.

## Requirements

* Unity 5.x.x

## Clone and Run

Clone into PenguinRun directory.

```
$ git clone <url/to/penguinrun.git> PenguinRun
```

After cloning, open and run with Unity.

## How to Play

Use arrow keys to move. Use space key to change perspective.
